# Fortune Avenue

A polished, self-contained property-trading board game for 2–4 local players.

Open `index.html` in a browser, or run a local server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Included

- Full animated 40-space board
- Property buying, color sets, houses, hotels, and escalating rent
- Stations, utilities, taxes, Chance, and Community Chest
- Doubles, jail, bankruptcy, and winner detection
- Automatic local save and resume
- Responsive desktop and mobile layout

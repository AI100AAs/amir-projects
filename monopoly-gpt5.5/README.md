# Metro Mogul

A self-contained local property-trading board game for 2–4 players.

Open `index.html` in a browser, or run a local server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Included

- Full 40-space board
- Local multiplayer for 2–4 players
- Property purchases, ownership, rent, taxes, stations, and utilities
- Chance and Community Fund events
- Jail, bankruptcy, and win detection
- Responsive desktop and mobile layout

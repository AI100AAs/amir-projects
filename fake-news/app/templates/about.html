About Truth Lens

What It Does

Truth Lens is a Fake News Detection System that analyzes news articles for credibility. It uses a pipeline of AI techniques to extract factual claims, cross-reference them against a curated knowledge base, and produce an overall credibility score with detailed reasoning.

Architecture

┌─────────────┐     ┌──────────────────┐     ┌────────────┐     ┌──────────────────┐
│   Input     │────→│  Claim Extraction │────→│    RAG     │────→│  Verdict & Score  │
│ Article/URL │     │  ┌──────────────┐ │     │  Retrieval │     │  ┌────────────┐  │
└─────────────┘     │  │Local (split) │ │     └────────────┘     │  │Similarity  │  │
                    │  │  or         │─┼─→  ChromaDB +   ──────→│  │  or        │  │
                    │  │LLM (API key)│ │     embeddings         │  │LLM judge   │  │
                    │  └──────────────┘ │                       │  └────────────┘  │
                    └──────────────────┘                       └──────────────────┘
  1. Input — Accepts article text or URL. URLs are fetched and parsed via newspaper4k.
  2. Claim Extraction — Local mode (no key): splits text into sentences and filters for claim-like statements. LLM mode (with API key): uses an LLM to identify 3–7 key factual claims.
  3. RAG Retrieval — Each claim is embedded (via sentence-transformers) and compared against a ChromaDB vector store seeded with verified claims from fact-checking sources.
  4. Verdict & Score — Local mode: uses semantic similarity distance to the closest matching claim. LLM mode: an LLM judge adjudicates each claim as True, False, or Misleading with reasoning.
  5. Aggregation — Claim-level verdicts are aggregated into an overall credibility score (0–100%) with a summary.

Features

Copy as JSON or download as Markdown report.

Persistent history with search, delete, and clear. Survives server restarts.

See progress live as each claim is extracted and evaluated via SSE.

🌱 Local Mode
Works out of the box, no keys needed. Uses sentence splitting + similarity search against 25 seed claims in ChromaDB.
⚡ LLM Mode
Enter any OpenAI-compatible API key (OpenRouter, OpenAI, etc.) in the UI for AI-powered extraction and reasoning. Key is used per-request only, never saved.

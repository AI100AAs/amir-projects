Why Argentina one run and France the next?

This is a probabilistic simulator, not a single fixed forecast. Football is noisy — especially finals between evenly matched teams (Argentina ~2148 Elo, France ~2134).

  • Quick sim (1 run) — rolls the dice once per match. Injuries, red cards, random refs, ±25 Elo jitter. Champion can flip every click. Good for “what if?” storylines.
  • Monte Carlo (default 100,000 runs/match) — each knockout tie is simulated 100k times; the majority winner advances. Match win % stabilises (e.g. 54% / 46%), but narrative randomness still runs inside each worker (refs, jitter, injuries, cards). Re-running can still change the bracket champion because later rounds inherit fatigue, suspensions, and morale from earlier stochastic results.
  • Bracket champion ≠ tournament odds — the trophy box shows one path through the bracket. The Champion odds panel runs 3,000 independent full-tournament sims and ranks who wins the cup most often.

Calibration data (pre-tournament)

Team attack/defence strengths are fitted from weighted match history. Live knockout predictions use live mode (includes 2026 results); the Accuracy tab uses pretournament only (no leakage).

  • World Cup 2018 & 2022 weight 1.0
  • Euro / Copa / AFCON / Asian Cup / qualifiers 2023–25 weight 0.58
  • World Cup 2026 completed matches weight 2.5 (live only)
  • Teams with sparse history Elo + squad priors

Blended xG engine

Expected goals for each team combine three signals multiplicatively on top of the Elo base:

  • Elo → xG share 70% blend
  • Dixon-Coles fitted form ×^0.12
  • Squad ratings (11 players) ×^0.22

Quick sim uses shot-by-shot micro-simulation (~9 shots per xG). Monte Carlo uses blended λ + Poisson (faster, same calibration).

Logistic win expectancy

xG split uses logistic Elo expectancy:

We1 =
11 + 10-ΔELO / 400

ELO = 55% World Elo + 45% squad composite. Close games (|ΔElo| < 220) compress xG toward parity to reflect draw/upset rates.

Dixon-Coles Poisson

Scorelines sampled from Poisson with correlation correction ρ = -0.20 (draws boosted). Defensive matchups (avg DEF+GK ≥ 74) shave ~0.55 xG.

P(A, B) = P(A; λ1) · P(B; λ2) · τ(A, B)

Capped at 5 goals per side. Group base 2.62 xG; knockout 2.28 xG before tactical mods.

Tactical & stage xG

  • 5-3-2 (Park the Bus) -0.35 xG
  • 4-4-2 (Structured) -0.15 xG
  • 4-3-3 (Attacking) +0.25 xG
  • Both defensive formations -0.20 xG, ρ → -0.10
  • Quarter-finals -0.08 xG
  • Semi-finals -0.15 xG
  • Final -0.20 xG

Extra time & penalties

Knockout draws → 30 min ET at 25% of regulation xG, then penalties.

xGET = xG90 × 0.25

Penalties use a historical shootout model (17 WC/Euro/Copa/AFCON shootouts): per-team conversion skill + GK quality + composure. Close Elo gaps regress toward 50/50. Base conversion ~75.5%.

Simulation modes

  • Runs per match default 100,000
  • Monte Carlo random ref, full narrative modifiers
  • Quick sim same modifiers, 1 run + shot micro-sim
  • Champion odds 3,000 full-bracket sims
  • Backtest neutral venue, no narrative mods

Climate, venue & travel

  • Host nation (home city) +28 ELO
  • Co-host / Americas at host venues +10–16 ELO
  • Adapted climate +20 ELO
  • Hostile climate -35 ELO
  • Mexico City altitude -25 ELO
  • Travel -5 ELO / 1000 km

Narrative modifiers

Active in Quick sim and Monte Carlo bracket runs. Disabled only in backtest so accuracy metrics stay fair.

  • Tactical counter +12 ELO
  • Big win morale +10 ELO
  • H2H mental edge +8 ELO
  • Cinderella story +12 ELO
  • Upset jitter ±25 ELO
  • Random injuries / reds / VAR probabilistic
  • ET / pen fatigue cascade up to -60 ELO

Accuracy evaluation

Honest backtest on 88 completed 2026 matches using pre-tournament calibration only (~68–70% outcome accuracy at 1,000 runs/match). Metrics: Brier score, log loss, group vs R32 split, biggest misses. Run node eval-backtest.mjs 1000 locally for comparison.
